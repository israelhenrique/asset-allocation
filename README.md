# asset-allocation

## Commands:

### Setup
```
npm install
```

### Running:
```
npm start
```
or
```
npm run start
```


### Verify lint:
```
npm run lint
```

### Try to auto fix lint:
```
npm run lint:fix
```

### Running tests:
```
npm run test
```

### Continuous testing on file changes:
```
npm run test:watch
```
